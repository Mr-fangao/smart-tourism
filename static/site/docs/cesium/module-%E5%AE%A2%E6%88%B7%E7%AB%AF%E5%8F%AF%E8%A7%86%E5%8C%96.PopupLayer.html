弹出窗图层

Fx构造函数

# new PopupLayer(map, container)

CesiumZondy.zondy.PopupLayer cesium的popup的实现

参数:
名称 类型 属性 默认值 描述
map Viewer

传入的cesium的地图对象viewer
position.entity Entity <optional>

实体,内部获取坐标点cartesian, 输入此参数可忽略下面的，cartesian，longitude,latitude
position.cartesian Cartesian3 <optional>

笛卡尔积坐标点cartesian, 输入此参数可忽略下面的longitude,latitude
position.longitude Number <optional>

经度, 先判断cartesian是否存在，存在忽略此参数
position.latitude Number <optional>

纬度, 先判断cartesian是否存在，存在忽略此参数
position.height Number <optional>

高度, 可选 默认0
options.popupId String <optional>

本次popup对应的唯一id,不传随机生成
options.popupContentId String <optional>

本次popup对应的唯一内容id
options.postRender Boolean <optional>
 true

是否实时渲染
options.showClose Boolean <optional>
 true

是否显示关闭按钮
options.callback Object <optional>
onShow function <optional>

显示popup事件的回调
onHide function <optional>

隐藏popup事件的回调
container Element | String

外部传入的div的字符串描述方式，一般是文字或者echarts的div;

  • 基础平台/创新中心 潘卓然

Example
这里唯一要注意的是我们中地数码的ceisum的右键事件不是放大缩小而是旋转视角
var cartesian1 = Cesium.Cartesian3.fromDegrees(
          100.108861,
          27.871516,
          0
        );
        var cartesian2 = Cesium.Cartesian3.fromDegrees(
          90.108861,
          37.871516,
          0
        );
        var cartesian3 = Cesium.Cartesian3.fromDegrees(
          116.108861,
          30.871516,
          0
        );
        popups = [cartesian1, cartesian2, cartesian3].map((c) => {
          const popup = new CesiumZondy.Overlayer.PopupLayer(
            webGlobe.viewer,
            {
              cartesian: c,
            },
            {
              popupId: "cesium-popup-id-1", //要保证唯一性
              popupContentId: "cesium-popup-content-id-1", //要保证唯一性
              postRender: true, //是否实时刷新
              showClose: false,
            },
          '<div id="echarts_id" style="height:100px;width:200px;color:#fff;">echats内容</div>'
        );
        return popup;
      });
