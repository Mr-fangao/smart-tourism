showBoundingVolume Boolean <optional>
 false

是否显示包围盒
maximumScreenSpaceError Number <optional>
 16

用于控制模型显示细节 值较大将会渲染更少的贴图,进而可以提高性能,而较低的值将提高视觉质量
options.layers String <optional>
