# applyEdits(edits, options)

要素编辑

参数:
名称 类型 属性 描述
edits Object

必选项，要执行的要素编辑操作。
addFeatures Array <optional>

可选项。添加要素的集合。格式addFeatures:[feature-object1,feature-object2,...]。 feature-object没有必填项，也就是说空值也会新增。[{},{}]如此则新增两个空数据。其中OBJECTID是自增的，无法指定。 格式为： [ { "geometry":{}, "attributes":{} } ] 参考文档：https://developers.arcgis.com/documentation/common-data-types/feature-object.htm
updateFeatures Array <optional>

可选项。要更新要素的集合。格式updateFeatures:[feature-object1,feature-object2,...]。 OBJECTID必须填写，其他选填。 格式为： [ { "geometry":{}, "attributes":{ "OBJECTID":"Your-ID",//OBJECTID必填，其他的都是选填。 ... } } ] 参考文档：https://developers.arcgis.com/documentation/common-data-types/feature-object.htm
deleteFeatures String <optional>

可选项。删除要素。
deleteFeatures.objectIds String <optional>

可选项。通过ObjectId来删除要素。
deleteFeatures.where String <optional>

可选项。通过where语句来删除要素。
deleteFeatures.geometry String <optional>

可选项。通过geometry对象来删除要素。
deleteFeatures.geometryType String <optional>

可选项。指定geometry的想的类型。
deleteFeatures.inSR String <optional>

可选项。指定geometry对象的坐标系，默认为地图坐标系。
deleteFeatures.spatialRel String <optional>

可选项。指定geometry与图层的相交关系，例如包含、相交、相离等，默认值为esriSpatialRelIntersects。 可选值为：esriSpatialRelIntersects | esriSpatialRelContains | esriSpatialRelCrosses | esriSpatialRelEnvelopeIntersects | esriSpatialRelIndexIntersects | esriSpatialRelOverlaps | esriSpatialRelTouches | esriSpatialRelWithin Example：spatialRel:esriSpatialRelIntersects
options Object

可选项，额外编辑选项。
gdbVersion String <optional>

可选项，地理数据库版本号。
rollbackOnFailure String <optional>

可选项，是否允许回滚，默认为true，可选值true|false。
useGlobalIds String <optional>

可选项，是否允许使用GlobalId，可选值true|false。

# queryFeatures(query)

查询要素信息

参数:
名称 类型 属性 描述
query String

必选项，查询参数。
geometry String <optional>

可选项。几何对象，要素查询条件，形式为x,y坐标。支持单个点，多个点、线、矩形、多边形。无需配合where使用，可单独查询。 格式为：geometry对象或xmin,ymin,xmax,ymax点坐标；Example：单个点：geometry=-104,35.6；封闭矩形：geometry=-104,35.6,-94.32,41；geometry对象：{xmin: -104, ymin: 35.6, xmax: -94.32, ymax: 41}； 参考链接：Geometry objects：https://developers.arcgis.com/documentation/common-data-types/geometry-objects.htm
geometryType String <optional>

可选项。配合geometry使用，当通过几何对象进行要素查询时，集合对象的类型，默认为矩形。可选值为：esriGeometryPoint | esriGeometryMultipoint | esriGeometryPolyline | esriGeometryPolygon | esriGeometryEnvelope Example：geometryType=esriGeometryEnvelope
spatialRel String <optional>

可选项。空间关系。即所选的几何对象与图层的相交关系。例如包含、相交、相离等，默认值为esriSpatialRelIntersects。 可选值为：esriSpatialRelIntersects | esriSpatialRelContains | esriSpatialRelCrosses | esriSpatialRelEnvelopeIntersects | esriSpatialRelIndexIntersects | esriSpatialRelOverlaps | esriSpatialRelTouches | esriSpatialRelWithin Example：spatialRel=esriSpatialRelIntersects
returnGeometry String <optional>

可选项。返回要素集合时，是否返回几何信息。默认为true。返回几何信息。可选值：true | false
geometryPrecision String <optional>

可选项。指定返回的要素集合中x、y坐标的小数位数。Example：geometryPrecision=3
maxAllowableOffset String <optional>

可选项。简化返回的几何要素时，允许的最大偏移量。对返回的要素的geometry对象起作用。
where String <optional>

可选项。通过where进行条件查询。与整个地图的查询相比没有返回值过滤。Example：where=POP2000 > 350000 参考链接：SQL语句：https://developers.arcgis.com/rest/services-reference/query-feature-service-.htm#ESRI_SECTION2_07DD2C5127674F6A814CE6C07D39AD46
sqlFormat String <optional>

可选项。sql语句的格式，默认为none。
objectIds String <optional>

可选项。通过Id查询要素。格式：objectIds=, ；Example：objectIds=37, 462
orderByFields String <optional>

可选项。返回的要素几何以何种方式进行排序。是升序或降序。格式：orderByFields=field1 , field2 , field3 ； Example： orderByFields=STATE_NAME ASC, RACE DESC, GENDER
groupByFieldsForStatistics String <optional>

可选项。"以某个字段进行分组。只有当outStatistics 为true是有效 格式：groupByFieldsForStatistics=field1, field2；Example：groupByFieldsForStatistics=STATE_NAME, GENDER"
outFields String <optional>

可选项。指定要显示的返回字段，为空时都不显示。Example：outFields=AREANAME,ST,POP2000；outFields=*
inSR String <optional>

            queryParamsLayer.objectIds = "155";
            queryParamsLayer.outFields = "*";
            //示例二 =====> where查询，这里我查询Name为金口街道的要素，并返回该要素的Name,countyname字段
            queryParamsLayer.where = "Name = '金口街道'";
            queryParamsLayer.outFields = "Name,countyname";
            //示例三 =====> geometry查询--简单点查询（封闭矩形），并设置returnCountOnly为true只返回要素数量
            queryParamsLayer = queryServiceByLayer.createQuery();

            queryParamsLayer.returnCountOnly = true;

            queryParamsLayer.geometry = {

                               [114.100,30.399]]],
                           "spatialReference" : {"wkid" : 4326}

            queryParamsMap.geometryType = "esriGeometryPolygon";
